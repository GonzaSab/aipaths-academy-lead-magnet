# Kit de Prompts para IA

Una coleccion curada de mas de 50 prompts listos para produccion, organizados por caso de uso. Copia, pega y obtiene resultados.

## Que Incluye

### Herramientas Base
- **prompt-builder.md** - Asistente interactivo que te guia para crear cualquier prompt personalizado
- **CLAUDE.md** - Configuracion de proyecto para flujos de prompt engineering

### Desarrollo de Software (3 archivos, 15+ prompts)
- Generacion de funciones y componentes
- Creacion de endpoints API
- Operaciones CRUD, flujos de autenticacion
- Debugging y resolucion de errores
- Generacion de tests y code review

### Operaciones de Negocio (3 archivos, 20+ prompts)
- Analisis de datos y dashboards KPI
- Sintesis de investigacion y resumenes
- Desglose de tareas y planificacion de sprints
- Agendas de reuniones y reportes de estado

### Marketing y SEO (3 archivos, 15+ prompts)
- Outlines de blog y meta descripciones
- Calendarios de contenido para redes sociales
- Secuencias de email y campanas
- Copy para anuncios en Meta, Google, LinkedIn

### Investigacion (1 archivo)
- Research Wizard - investigacion guiada con reportes estructurados

### YouTube (2 archivos)
- Asistente de investigacion de titulos con optimizacion SEO
- Generador de metadata para Shorts

### Automatizacion N8N (1 archivo)
- Prompt para construir workflows en n8n via MCP

## Como Usar

1. Navega la categoria que corresponda a tu tarea
2. Encuentra la plantilla de prompt que necesitas
3. Reemplaza los `[PLACEHOLDERS]` con tus detalles especificos
4. Pega en ChatGPT, Claude, o cualquier asistente de IA
5. Itera sobre el resultado segun sea necesario

## Niveles de Calidad de Prompts

Cada prompt esta estructurado para uso en produccion:

```text
Role: Quien debe actuar la IA
Task: Objetivo claro y especifico
Context: Informacion de contexto
Requirements: Restricciones y especificaciones
Format: Estructura del output deseado
```

## Tips para Mejores Resultados

- Completa TODOS los placeholders - se especifico
- Empieza con prompts simples, agrega complejidad segun necesites
- Usa el Prompt Builder wizard para necesidades personalizadas
- Combina prompts para flujos de trabajo complejos
