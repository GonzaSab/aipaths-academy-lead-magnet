## How to Use

1. Browse the category that matches your task
2. Find the prompt template you need
3. Replace `[PLACEHOLDERS]` with your specific details
4. Paste into ChatGPT, Claude, or any AI assistant
5. Iterate on the output as needed

## Prompt Quality Levels

Each prompt is structured for production use:

```text
Role: Who the AI should act as
Task: Clear, specific objective
Context: Background information
Requirements: Constraints and specifications
Format: Desired output structure
```

## Tips for Best Results

- Fill in ALL placeholders - be specific
- Start with simpler prompts, add complexity as needed
- Use the Prompt Builder wizard for custom needs
- Combine prompts for complex workflows
