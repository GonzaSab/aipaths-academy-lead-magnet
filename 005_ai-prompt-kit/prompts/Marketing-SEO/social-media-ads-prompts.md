# Social Media & Advertising Prompts

> Free prompts for social media content and paid advertising. Sources: [WebFX](https://www.webfx.com/blog/ai/ai-prompt-examples/), [StoryChief](https://storychief.io/blog/ai-marketing-prompts), [Juma](https://juma.ai/blog/chatgpt-prompts-for-marketing)

## Social Media Content

### Platform-Specific Posts
```
Create 5 social media posts about [TOPIC/PRODUCT] for [PLATFORM].

Platform specs:
- LinkedIn: Professional, 150-300 words, thought leadership
- Twitter/X: Punchy, under 280 chars, thread-worthy
- Instagram: Visual-first, caption 125-150 words, hashtag suggestions
- TikTok: Trend-aware, casual, hook-focused

Include:
- Hook/opening line
- Main message
- Call-to-action
- Relevant hashtags (for IG/TikTok)
```

### Content Calendar Generator
```
Create a 2-week social media content calendar for [BUSINESS TYPE].

Platforms: [PLATFORMS]
Posting frequency: [X posts per platform per week]

For each post include:
- Day and platform
- Content type (educational/promotional/engagement/entertaining)
- Topic/theme
- Caption draft
- Visual suggestion
- Best posting time

Balance: 80% value, 20% promotional
```

### Viral Hook Generator
```
Generate 10 scroll-stopping hooks for [TOPIC/NICHE] on [PLATFORM].

Types to include:
- Controversial opinion
- Surprising statistic
- Personal story opener
- Question that resonates
- "Most people don't know..."
- Challenge/call-out
- Before/after setup

Keep under 15 words each. High curiosity factor.
```

## Video Content

### Short-Form Video Script
```
Write a short-form video script (under 60 seconds) for [PLATFORM: TikTok/Reels/Shorts].

Topic: [TOPIC]
Goal: [educate/entertain/sell]

Structure:
- Hook (0-3 sec): Stop the scroll
- Setup (3-10 sec): Context
- Value/Story (10-45 sec): Main content
- CTA (45-60 sec): What to do next

Include:
- On-screen text cues
- Audio/music suggestions
- Visual transition notes
```

### YouTube Video Outline
```
Create a YouTube video outline for: "[VIDEO TITLE]"

Target length: [X] minutes
Audience: [TARGET AUDIENCE]

Include:
- Hook script (first 30 seconds)
- Chapter breakdown with timestamps
- Key points per section
- B-roll/visual suggestions
- End screen CTA
- Description draft with keywords
- 10 title alternatives
```

## Paid Advertising

### Facebook/Meta Ads
```
Write 3 Facebook ad copy variations for [PRODUCT/SERVICE].

Details:
- Offer: [OFFER DETAILS]
- Target audience: [DEMOGRAPHICS, INTERESTS]
- Goal: [leads/sales/awareness]

For each variation provide:
- Primary text (125 characters ideal)
- Headline (40 characters)
- Description (30 characters)
- CTA button recommendation

Angles: 1) Pain point, 2) Benefit-focused, 3) Social proof
```

### Google Ads Copy
```
Generate Google Ads copy for the keyword "[TARGET KEYWORD]"

Create:
- 5 headlines (30 characters max each)
- 3 descriptions (90 characters max each)
- Display URL path suggestions

Landing page: [URL/DESCRIPTION]
Unique selling points: [USP 1, 2, 3]
Target audience: [AUDIENCE]
```

### LinkedIn Ads
```
Create LinkedIn ad copy for [B2B PRODUCT/SERVICE].

Audience: [JOB TITLES, INDUSTRIES]
Campaign objective: [lead gen/awareness/conversions]

Provide:
- Intro text (150 words max)
- Headline options (3)
- CTA variations
- Targeting suggestions

Tone: Professional but not boring
Include: Specific pain point, clear value prop, credibility element
```

## Influencer & UGC

### Influencer Brief
```
Create a creative brief for influencer collaboration.

Brand: [BRAND NAME]
Product: [PRODUCT]
Campaign goal: [GOAL]
Platform: [PLATFORM]

Include:
- Key messages (3 max)
- Required mentions/hashtags
- Do's and don'ts
- Content format suggestions
- Talking points
- Disclosure requirements
```

### UGC Ad Script
```
Write a UGC-style ad script for [PRODUCT].

Format: Authentic, first-person testimonial
Length: 30-45 seconds
Platform: [PLATFORM]

Structure:
- Attention hook (problem/relatable moment)
- Discovery story
- Product demo/features
- Results/transformation
- Recommendation + CTA

Tone: Natural, unscripted feel
```
