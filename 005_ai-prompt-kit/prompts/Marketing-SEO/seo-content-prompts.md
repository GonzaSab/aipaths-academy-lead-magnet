# SEO Content Prompts

> Free prompts for SEO content creation and optimization. Sources: [HubSpot](https://blog.hubspot.com/marketing/ai-prompt-examples), [Glean](https://www.glean.com/blog/ai-prompts-for-marketing), [Founderpath](https://founderpath.com/blog/ai-prompt-marketing-business)

## Blog Content Planning

### Detailed Blog Outline
```
Create a detailed blog outline for a 1,500-word article targeting the keyword '[TARGET KEYWORD]'.

Include:
- H2 and H3 heading suggestions
- Potential statistics to research
- A compelling introduction approach
- Key points to cover in each section
- A strong conclusion with CTA

Target audience: [AUDIENCE]
Tone: [informative/conversational/professional]
```

### SEO Blog Post Generation
```
Write a 1,500-word blog post titled '[TITLE]'.

Requirements:
- Target audience: [AUDIENCE TYPE]
- Tone: informative and engaging
- Include these keywords naturally: [KEYWORD LIST]
- Structure with H2s and H3s
- Add a compelling introduction hook
- Include actionable takeaways
- End with a clear call-to-action
```

### Topic Ideas for Keywords
```
Suggest 10 blog post topics that would rank well for the keyword '[KEYWORD]'.

For each topic provide:
- Suggested title (60 characters max)
- Search intent (informational/transactional/navigational)
- Key points to cover
- Potential secondary keywords
```

## Meta Descriptions & Headlines

### Meta Description Generator
```
Write 3 SEO-friendly meta descriptions for a blog post about [TOPIC].

Requirements:
- 150-160 characters each
- Include the primary keyword: [KEYWORD]
- Include a clear value proposition
- Add urgency or curiosity element
- Include a subtle call-to-action
```

### Headline Optimization
```
Generate 10 alternative headlines for this article: '[CURRENT HEADLINE]'

Requirements:
- More attention-grabbing while maintaining SEO value
- Target keyword: '[TARGET KEYWORD]'
- Mix of formats: how-to, listicles, questions, power words
- Keep under 60 characters for SERP display
```

## Keyword Research

### Keyword List Generation
```
Generate a list of 20 high-traffic keywords related to [TOPIC] that we can target in upcoming content.

Organize by:
- Primary keywords (high volume, competitive)
- Long-tail keywords (lower volume, easier to rank)
- Question-based keywords (for featured snippets)
- Commercial intent keywords (for conversions)

Include estimated search intent for each.
```

### Competitor Keyword Gap Analysis
```
I'm in the [INDUSTRY] space. My main competitors are [COMPETITOR 1, 2, 3].

Suggest keywords they likely rank for that I should target, categorized by:
- Informational content opportunities
- Product/service page opportunities
- Comparison content opportunities
- FAQ content opportunities
```

## Content Optimization

### Existing Content Audit
```
Analyze this content for SEO improvements:

[PASTE CONTENT]

Provide:
- Keyword density assessment
- Suggestions for better keyword placement
- Header structure improvements
- Internal linking opportunities
- Content gaps to address
- Readability improvements
```

### Featured Snippet Optimization
```
Rewrite this paragraph to optimize for a featured snippet for the query '[QUERY]':

[PASTE PARAGRAPH]

Create versions for:
- Paragraph snippet (40-60 words)
- List snippet (numbered steps)
- Table snippet (if applicable)
```
