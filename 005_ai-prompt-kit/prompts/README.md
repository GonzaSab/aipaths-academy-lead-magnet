# AI Prompt Kit

50+ production-ready prompts organized by category.

## Quick Start

1. Browse the `prompts/` folder
2. Pick a category that matches your task
3. Open the relevant `.md` file
4. Copy a prompt, fill in the `[PLACEHOLDERS]`
5. Paste into your AI assistant

## The Prompt Builder

Not finding what you need? Use `prompt-builder.md`:

1. Copy the entire prompt into any AI
2. Answer the wizard's questions
3. Get a custom, production-ready prompt

## Tips

- **Be specific**: Replace all `[PLACEHOLDERS]` with real details
- **Iterate**: First output rarely perfect - refine and re-run
- **Combine**: Chain prompts for complex workflows
- **Save**: Keep your best customized versions
